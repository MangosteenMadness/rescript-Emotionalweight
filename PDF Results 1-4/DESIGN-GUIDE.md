## ‚ú® Design Best Practices

1. **White Space is Your Friend**
   - Don't cram content
   - At least 20px padding inside boxes
   - Generous margins between elements

2. **Hierarchy is Key**
   - Largest: Page titles
   - Medium: Section headers
   - Smallest: Body text
   - Consistent sizing throughout

3. **Visual Breaks**
   - Use boxes, lines, or space to separate sections
   - Alternate between full-width and boxed content
   - Break up long text with callouts

4. **Consistency**
   - Same header style on every page
   - Same footer on every page
   - Same colors for same element types
   - Same font sizes for same text levels

5. **Readability**
   - Dark text on light background
   - Minimum font size: 12pt
   - Line length: max 65-75 characters
   - Line height: 1.6 minimum

## üéØ Canva Tips

**Setting Up in Canva:**
1. Go to Canva.com
2. Click "Create a design"
3. Choose "Document" (8.5 x 11 inches)
4. Search templates: "Wellness Guide" or "eBook"
5. Choose one with clean, modern design
6. Customize colors to match palette above

**Efficient Workflow:**
1. Create page 1 (cover) first
2. Create page 2 (score page) - duplicate 4 times for each version
3. Create pages 3-5 (same for all versions)
4. Create pages 6-7 (prompts) - duplicate 4 times for each version
5. Create pages 8-10 (same for all versions)

**Exporting:**
1. Click "Share" ‚Üí "Download"
2. Choose "PDF Print" (highest quality)
3. Check "Flatten PDF" (prevents editing)
4. Download
5. Name files clearly:
   - emotional-weight-guide-light-load.pdf
   - emotional-weight-guide-moderate-weight.pdf
   - emotional-weight-guide-heavy-burden.pdf
   - emotional-weight-guide-overwhelming-weight.pdf
